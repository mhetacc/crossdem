\subsection{Token Count and Textual Complexity}

Overall the collected corpus amount of 473 speeches from De Gasperi, and 72 from Meloni, with a total token count of $609,142$ for De Gasperi and $120,116$ for Meloni. On average, Meloni's corpus have 1668 tokens per speech, while De Gasperi's 1287 tokens per speech. Results in table \ref{tab:politician_tokens}.

I then measured the textual lexical diversity using \textit{MTLD}, which I preferred over the more common \textit{TTR} (Type-Token Ratio) since MTLD is agnostic on the corpus size. I used the \textit{LexicalRichness} library\footnote{LexicalRichness is a small Python module to compute textual lexical richness (aka lexical diversity) measures: \url{https://pypi.org/project/lexicalrichness/}.} to compute the MTLD for each speech, and then both the average and the median MTLD for both PMs. Results can be seen in table \ref{tab:politician_mtld}.

To better visualize the difference of textual diversity I plotted the MTLD values of 70 different speeches of Meloni and De Gasperi against each other. The plot can be seen in figure \ref{fig:mtl_comparison}.
